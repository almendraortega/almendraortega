Funciones para el codigo

Codigo

Variables globales

(leaves) para almacenar instancias de copos de nieve, (leafTypes) para almacenar tipos de copos de nieve y slider para controlar el número de este.

Función setup

Se ejecuta al principio.En este caso, crea un lienzo (canvas) de 400x400 píxeles.

slider

Se crea un deslizador con un rango de 0 a 100 y un valor inicial de 50. slider = createSlider(0, 100, 50);

Entonces, este deslizador tiene un rango que va desde 0 hasta 100, y su valor inicial es 50.

Array

Se llena el array leaves con 100 instancias (moldes) de la clase Leaf, y se asignan tipos de copos aleatorios.

(leaf tiene propiedades como la posición, velocidad y tamaño de la hoja)

Función draw

Se establece el color de fondo.

Se itera a través del array leaves, haciendo que cada copo caiga y se muestre.

Se eliminan los copos que han caído fuera de la pantalla.

Se determina cuántos copos nuevas se deben agregar según el valor del deslizador y se agregan.
