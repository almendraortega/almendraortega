GLOSARIO

ARRAY

Un array, en programación, es una estructura de datos que almacena una colección ordenada de elementos La idea fundamental es que puedes almacenar varios valores bajo un mismo nombre y acceder a esos valores mediante su posición en el array.

INSTANCIAS

Es la creación de un objeto a partir de una clase. Una instancia puede tener o va a tener estado, comportamiento e identidad. El estado va a representar los valores de cada característica o atributo del objeto. es como un molde.

ITERACION

En programación, una iteración se refiere a la repetición de un conjunto de instrucciones o un bloque de código en un programa. Es una forma de ejecutar una serie de declaraciones varias veces, generalmente hasta que se cumple una condición específica.

TRUE

En programación, el valor true representa la afirmación de una condición o la activación de una característica.
