Funciones para el codigo

Codigo

Función setup

Se crea un arreglo llamado particulas.

Se establece el canvas para ocupar toda la ventana del navegador (windowWidth, windowHeight)

Se define la cantidad de partículas y se crea cada partícula con una posición y color aleatorio.

Function draw

se recorre el arreglo de partículas y se llama al método display() de cada una para mostrarlas en el lienzo.

Cambio de Color al hacer "Click"

Cuando se presiona el ratón (mousePressed()),

Se recorren todas las partículas y se llama al método cambiarColor(), que asigna un nuevo color aleatorio a cada partícula.

Display

Actualiza la posición, envuelve la partícula alrededor del lienzo y la dibuja.

Es la rutina que cada partícula sigue para actualizar su posición, asegurarse de que no se escape de la pantalla y mostrar su presencia en el lienzo.
